# JATE: Just Another Text Editor

## Description
This application a text editor that runs in the browser. This is a single-page application that meets PWA critera. It has data persistence and is functional offline. 

## Built With
* JavaScript 
* Webpack 
* IndexedDB
* idb.js
* Workbox
